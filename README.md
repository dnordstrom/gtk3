## Installation

```sh
  # First, clone the repository.
  $ git clone https://github.com/rxyhn/gtk3.git && cd gtk3

  # Then, install sass compiler.
  $ npm install -g sass
  
  # Compile and install the theme.
  $ make && sudo make install
```
